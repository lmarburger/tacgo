# tacgo

Time and track pomodori against the top task in [Taco].

[taco]: https://tacoapp.com

## Getting Started

It's still rough around the edges.

 1. Install the gem:

    ```
    $ gem install tacgo
    ```

 2. Add your Taco credentials to `~/.netrc`.

    ```
    machine tacoapp.com
      login larry@marburger.cc
      password xxxxxxxxxxxxxxxxxxxx
    ```

 3. Work!

   ```
   $ tacgo
   ```
